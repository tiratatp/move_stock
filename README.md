move_stock
==========

A program to move stock in Central Department Store, Thailand

Dependencies
____________
- xlrd
- ConfigParser

Usage
_____
- Windows
  1. Download repository
  2. Extract dist folder
  3. Open .xls file with move_stock.exe
- Unix
  1. Use pip to install xlrd, ConfigParser
  2. Download repository
  3. move_stock.py <.xls>
